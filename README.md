# 📌 Project Overview

This project is part of the **Quantium Data Analytics Virtual Experience Program** hosted by **Forage**.

The objective is to determine whether a new store layout generated a statistically significant uplift in sales by comparing selected trial stores with carefully matched control stores.

The project combines:

- Retail Analytics
- Experimental Design
- Statistical Analysis
- Data Visualization
- Commercial Business Insights

to simulate the responsibilities of a Retail Data Analyst at Quantium. Quantium applies AI and analytics to retail decision-making and customer intelligence at scale. :contentReference[oaicite:1]{index=1}

---

# 🎯 Business Problem

Quantium introduced a new store layout in three trial stores.

The business wants to know:

- Did the new layout increase sales?
- Did customer traffic improve?
- Were observed improvements statistically significant?
- Should the rollout continue across other stores?

This project answers these questions using data-driven experimentation.

The dataset contains historical retail transaction information including:

# 📈 Project Workflow

## Step 1

Data Preparation

- Load retail transaction data
- Convert date columns
- Aggregate monthly metrics

---

## Step 2

Pre-Trial Analysis

Calculated:

- Monthly Sales
- Number of Customers
- Transactions per Customer

---

## Step 3

Control Store Selection

Used two similarity measures:

- Correlation Score
- Magnitude Distance

Selected the most comparable control store for each trial store.

---

## Step 4

Statistical Testing

Performed significance testing to determine whether observed sales uplift exceeded the expected confidence interval.

---

## Step 5

Visualization

Created professional business charts showing:

- Monthly Sales
- Customer Count
- Trial vs Control Comparison
- Confidence Intervals
- Statistical Significance

These techniques help distinguish genuine business impact from normal variation.
